Ρυθμίσεις
=========

Στην οθόνη των ρυθμίσεων μπορείτε

- να βρείτε γενικές πληροφορίες σχετικά με τη συνδρομή
- να προσθαφαιρέσετε χρήστες που μπορούν να χρησιμοποιούν την εφαρμογή
- να προσθαφαιρέσετε τύπους ιδιοτήτων για πελάτες, οχήματα, εργασίες, ...
- να αλλάξετε τη γλώσσα εμφάνισης της εφαρμογής
