Προβολή
-------

Μπορείτε να δείτε τις εργασίες είτε όλων των οχημάτων όλων των πελατών
είτε ενός συγκεκριμένου οχήματος.

Εργασίες όλων των οχημάτων
~~~~~~~~~~~~~~~~~~~~~~~~~~

Για να ανοίξετε τη καρτέλα μιας εργασίας
μπορείτε να πάτε στη κεντρική λίστα των εργασιών
(περιέχει όλες τις εργασίες όλων των οχημάτων από όλους τους πελάτες)
είτε από την κεντρική οθόνη

.. figure:: /_static/images/screen-main-services.png

είτε από το κεντρικό μενού

.. figure:: /_static/images/screen-mainmenu-services.png

και να επιλέξετε το όχημα που επιθυμείτε

.. figure:: /_static/images/screen-services-view-select.png

Επίσης, μπορείτε να δείτε τη λίστα εργασίων
όλων των οχημάτων ενός συγκεκριμένου πελάτη.

:ref:`Δείτε εδώ πως <customer_service_view>`.

Εργασίες ενός συγκεκριμένου οχήματος
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

:ref:`Δείτε εδώ πως <vehicle_service_view>`.

Αναζήτηση
---------

Εκτός από γενική αναζήτηση

.. figure:: /_static/images/screen-searchfield.png

μπορείτε να κάνετε και εξειδικευμένη αναζήτηση πατώντας το κουμπί "Αναζήτηση"

.. figure:: /_static/images/screen-search-button.png

και στη συνέχεια κάνοντας μια επιλογή.

.. figure:: /_static/images/screen-services-search-options.png

Ημ/νία έναρξης & λήξης
~~~~~~~~~~~~~~~~~~~~~~

Για να περιορίσετε το εύρος της ημ/νίας συμπληρώστε ένα από ή/και όλα τα πεδία

- "ΑΠΟ"
- "ΕΩΣ"

.. figure:: /_static/images/screen-services-search-options-datetime.png

:DD: Ημέρα
:MM: Μήνας
:YYYY: Έτος
